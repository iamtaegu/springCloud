# springCloud
스프링 클라우드 코딩 공작소 (2판)

## 마이크로서비스 애플레키엿ㄴ 구축 기술 
 * 스프링 클라우드
 * 스프링 부트
 * 스웨거
 * 도커
	* 지속적으로 출시하고 통합할 수 있는 자체 빌드 및 배포 파이프라인 구축 
 * 쿠버네티스
 * ELK 스택
 * 그라파나
 * 프로메테우스
 
## 스프링 부트 
 * 스프링 프레임워크를 재구성한 것인데
 * 스프링의 핵심 기능은 수용하고, 불필요한 엔터프라이즈 제품은 걷어냈음
 * 자바 기반의 REST 지향 프레임워크를 제공하며
 * 애너테이션 기반으로 패키징하고 배포할 수 있는 REST 서비스를 신속히 만들 수 있게 함
